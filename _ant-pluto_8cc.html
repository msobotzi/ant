Detailed Description

Pluto generator forntend.

Two modes available: Single reaction mode and Random Gun

Random gun: Run with –gun to shoot particles of a selected type uniformly in all directions.

Reaction Mode: Generate physics using pluto by specifiying a reaction string. Run with –pluto –reaction "...." Where "...." is a Pluto reaction string, for example "p omega [ pi0 g ]": for omega production followed by a decay into pi0 gamma.

To further decay the particles use –enableBulk. Then all instable particles decay according to the database.
